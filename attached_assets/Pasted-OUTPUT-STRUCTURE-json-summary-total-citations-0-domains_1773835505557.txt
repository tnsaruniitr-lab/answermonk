how_they_appear must be verbatim language — not a summary of what the source says but the actual words used. If web search is active copy quoted phrases directly from the page
tactic titles must be action verb phrases — never category labels. Write what brands DO not what the content type is

actions array must contain exactly one entry per brand — mandatory, do not end without it
quick_win must reference a real domain and real citation count from the CSV
