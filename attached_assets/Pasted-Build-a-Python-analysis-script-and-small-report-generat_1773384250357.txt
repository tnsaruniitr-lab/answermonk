Build a Python analysis script and small report generator for the healthcare GEO citation CSV already in the project.

Goal:
Calculate brand-level metrics for 6 main brands:
- Emirates Home Nursing
- First Response Healthcare
- Nightingale Health Services
- Vesta Care
- Manzil Health
- Call Doctor

Important:
- Work from the raw snippet/context data in the CSV.

Deliverables:
1. A Python script `analyze_geo_healthcare.py`

Metrics to compute:

A) Authority Source Presence
Definition:
- Presence on repeated winner-source network and external source spread.
Use:
- distinct hosts per brand
- distinct domain_category per brand
- external vs own-site presence
- repeated winner hosts across all 6 brands
Formula:
- 40% repeated winner-host presence
- 25% host diversity
- 20% domain category diversity
- 15% external share
Return 0-100 normalized score.

B) Category Clarity
Search in `context` for category/provider framing keywords:
- at-home healthcare
- in-home healthcare
- home healthcare
- care at home
- home care
- home care provider
- at-home care provider
- home nursing provider
- healthcare provider
- provider in dubai
- based in dubai
Formula:
- snippet prevalence = matched deduped snippets / total deduped snippets
- domain prevalence = matched distinct hosts / total distinct hosts
- final = 0.7 * snippet prevalence + 0.3 * domain prevalence
Return 0-100.

C) Trust
Search in `context` for:
- dha

- professional

- doctor-led

- trained caregivers

Formula:
- same prevalence logic as Category Clarity
- final = 0.7 * snippet prevalence + 0.3 * domain prevalence
Return 0-100.

D) Service Breadth
Create 4 service buckets from `context`:
1. home nursing
2. physiotherapy
3. doctor support
4. diagnostics / blood tests
Use robust keyword matching for each bucket.
For each bucket:
- snippet prevalence
- domain prevalence
Bucket score = 0.7 * snippet prevalence + 0.3 * domain prevalence
Final breadth score = average of 4 bucket scores
Return 0-100.

H) Identity Consistency
Build a packet repeatability score from:
- category clarity markers
- trust markers
- breadth bucket presence
- care depth markers
For each deduped snippet, build a boolean packet vector.
For each brand:
- dominant packet strength = prevalence of top recurring packet combination
- cross-domain packet stability = how often dominant packet appears across distinct hosts
- drift penalty = 1 - normalized entropy of packet combinations
Final:
- 50% dominant packet strength
- 30% cross-domain packet stability
- 20% inverse drift
Return 0-100.
